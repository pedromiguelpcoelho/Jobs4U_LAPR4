REQUIREMENT 1
SHORT_ANSWER_QUESTION: What is the difference between an Interface and an Abstract class?
ANSWER: An interface is a reference type in Java that is similar to a class. It is a collection of abstract methods. An abstract class is a class that cannot be instantiated and is designed to be subclassed by other classes. The main difference between an interface and an abstract class is that an interface can only contain abstract methods, while an abstract class can contain both abstract and concrete methods.
